크로믹시란?

고품질 AI 학습데이터 구축을 위한 크라우드 소싱 플랫폼

크라우드 소싱은 'crowd'와 외부 발주를 의미하는 'outsourcing'의 합성어로 대중을 제품이나 창작물 생산 과정에 참여시키는 방식을 말합니다.

크로믹시에서는 수백명 이상의 크라우드워커들이 쉽게 협업하고 프로젝트를 수행하여 생성된 데이터는 인공지능 산업생태계를 조성하는데 기여합니다.

# 이미지생성

[음성레이블링도구]


음성레이블링은 음성 데이터를 처리하는 작업을 말합니다. 이 작업에서는 음성 데이터에서 발화되는 부분을 정확히 찾아내고, 각 발화의 구간과 누가 발화하는지와 같은 정보를 표시하는 작업을 수행합니다. 이를 통해 음성 데이터의 내용을 더 쉽게 이해하고 활용할 수 있도록 합니다.

[이미지레이블링도구]


이미지나 영상에서 특정 물체나 사물의 영역을 정확히 찾아 영역 추출한 후에, 추출한 영역 안에 있는 물체를 명확하게 표시해주는 어노테이션을 합니다.

[3D Point Cloud 레이블링 도구]


3D Point Cloud 레이블링 도구는 3차원 공간에 퍼져 있는 점들로 구성된 Point Cloud 데이터에 대해 객체를 식별하고 어노테이션하는 도구를 말합니다. 이러한 도구는 컴퓨터 비전 및 로봇 공학 분야에서 널리 사용되며, 다양한 응용 분야에서 3D 환경에서 객체를 분류하고 인식하는 데 활용됩니다.
