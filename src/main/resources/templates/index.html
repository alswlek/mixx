작업 시작하기

PROJECT

작업자는 cromixi에서 제공하는
다음과 같은 형태의 데이터
수집 · 가공 작업을 진행합니다.

텍스트

텍스트 데이터를 음성 데이터로 변환할때 필요한 데이터를 가공합니다.

이미지

인공지능 학습을 위해 이미지 데이터를 수집합니다. Bounding Box, Semantic Segmentation, 속성분류 등의 가공 작업을 진행합니다.

영상

인공지능 학습을 위해 영상 데이터를 수집합니다. 프레임추출, 속성 분류 등의 데이터 가공 작업을 수행합니다.

음성

음성 데이터를 문자 및 다양한 형태의 데이터로 변화하기 쉽도록 음성데이터를 가공합니다.

"데이터 라벨링이란?"

데이터 라벨링이란, 인공지능이 특정 데이터를
스스로 학습할 수 있도록 데이터를 알맞게
수집 및 가공하는 작업을 말합니다.


작업자 : 데이터를 수집 및 가공합니다.

검수자 : 데이터가 올바르게 수집 및 가공되었는지 확인합니다.
